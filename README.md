# Functional Programming in JavaScript

## Overview

This project contains practical implementations of the core Functional Programming concepts in JavaScript. Each topic is implemented in a separate file to demonstrate the concepts clearly and independently.

## Topics Covered

* Declarative vs Imperative Programming
* Why Functional Programming?
* Memoization
* Immutability & Side Effects
* Pure Functions
* Function Composition
* Higher-Order Functions
* Currying
* Lodash Utilities
* Ramda Functional Pipelines

## Installation

Clone the repository and install the required dependencies.

```bash
npm install
```

Install Lodash and Ramda if they are not already installed.

```bash
npm install lodash ramda
```

## Running the Programs

Run each file individually using Node.js.

```bash
node src/01-declarative-imperative.js
node src/02-why-functional-programming.js
node src/03-immutability-side-effects.js
node src/04-pure-functions.js
node src/05-function-composition.js
node src/06-higher-order-functions.js
node src/07-currying.js
node src/08-lodash-ramda.js
```

## Learning Outcomes

After completing this project, you will be able to:

* Differentiate between imperative and declarative programming.
* Understand and implement pure functions.
* Identify and avoid side effects.
* Apply immutable programming techniques.
* Build reusable higher-order functions.
* Implement function composition using `pipe` and `compose`.
* Create curried functions and perform partial application.
* Use Lodash and Ramda to simplify functional programming tasks.
